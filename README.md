##  1. Objetivos

Este projeto propõe um protótipo de um sistema de monitoramento de segurança interno para aeronaves. A principal motivação é aplicar os conceitos de IoT para desenvolver um modelo funcional que resolva problemas do mundo real, focando em um sistema crítico para a segurança e preservação de veículos aéreos.
